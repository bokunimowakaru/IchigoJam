■ メモリマップ(IchigoJam相対アドレス)

    #000-#6FF キャラクタ領域（固定）
    #700-#7FF キャラクタ領域（書き込み可能）
    #800-#8FF 変数領域
    #900-#BFF VRAM領域
    #C00-#FFF プログラム領域

■ IchigoJam BASIC コマンド

    「HELP」にてメモリマップ表示が可能
    (但し、Ver 1.1で削除される可能性がある)

■ 使用例

    fori=#C00to#C1F:?hex$(i,3),chr$(peek(i)),peek(i):next

■ 相対アドレスの基準ポインタ

    マシン語呼び出しの第二パラメータにMEM#000の基準ポインタ
    (IchigoJam BASIC Version 1.1からの対応予定)

            #include 

            int16_t usr_calc(int16_t val, char* mem) {
                int i;
                for (i = 0; i < 32 * 24; i++) {
                    mem[i + 0x900] = val;
                }
                return 0;
            }
